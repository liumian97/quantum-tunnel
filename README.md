# quantum-tunnel

#### 介绍
QuantumTunnel，量子隧道，名字来源于量子纠缠现象。 > 两个处于量子纠缠的粒子，无论处于多么远的距离，当其中一个粒子状态改变时，另外一个粒子也会做出相应的改变。

QuantumTunnel也取意于此，希望把公网发出来的请求，完整的同步到内网，就像在内网发出的请求，打破网络的限制。
